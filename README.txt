== Description ==

Confetti on Post Update est un plugin qui ajoute une animation festive lorsque vous mettez à jour ou publiez un article dans WordPress. Que vous utilisiez l'éditeur classique ou Gutenberg, ce plugin vous permet de célébrer chaque sauvegarde avec des confettis !

Fonctionnalités principales :
* Lance des confettis lorsque vous cliquez sur "Mettre à jour" ou "Publier" dans l'éditeur.
* Compatible avec Gutenberg et l'éditeur classique.
* Léger et sans impact sur la performance du site.

== Installation ==

1. Téléchargez le fichier zip du plugin et décompressez-le.
2. Téléchargez le dossier `confetti-on-post-update` dans le répertoire `/wp-content/plugins/`.
3. Activez le plugin depuis le menu "Extensions" dans WordPress.
4. Profitez des confettis lors de la publication de vos posts !

== Frequently Asked Questions ==

= Est-ce que le plugin fonctionne avec Gutenberg ? =

Oui, il fonctionne à la fois avec l'éditeur Gutenberg et l'éditeur classique.

= Puis-je personnaliser les confettis ? =

Pas pour le moment, mais des options de personnalisation pourraient être ajoutées dans une future version.
